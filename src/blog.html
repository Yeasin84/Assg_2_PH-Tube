Discuss the scope of `var`, `let`, and `const`

In JavaScript, `var` has a function-level scope, while `let` and `const` have block-level scope. `let` allows you to reassign a variable's value, while `const` declares a constant whose value cannot be changed after initialization.

Tell us the use cases of `null` and `undefined`

`null` is often used to intentionally represent the absence of any object value, while `undefined` typically represents an uninitialized variable or a function that has no return value.

What do you mean by REST API?

REST (Representational State Transfer) is an architectural style used in web development. A REST API (Application Programming Interface) is a set of rules and conventions for building and interacting with web services following REST principles.
